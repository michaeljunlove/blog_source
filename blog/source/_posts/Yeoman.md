# yeoman介绍

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;yeoman是一个脚手架工具

# 安装yeoman

``` javascript
npm install -g yo
```

# 查看yeoman版本

``` javascript
yo --version
```


# 查看本地安装了哪些脚手架


``` javascript
npm ls -g --depth=1 2>/dev/null | grep generator-
```


# 用脚手架生成项目

``` markdown
yo react-webpack galleryByReact
```
